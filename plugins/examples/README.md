These examples are created to show what plugin should look like and its capabilities

>To use example plugins, drag plugin corresponding to your operating system into plugins directory
>

### Explanation

* **shutdown** - this plugin turns off computer

> Line 2 must be a comment describing plugin

* **screenshot** - takes a screenshot, puts it in temp (in the case of Windows) or tmp (in the case of Linux) folder and sends it to the bot

> To send exe, jar, zip, etc., you must specify doc
>

* **process** - the plugin checks whether the process is running
